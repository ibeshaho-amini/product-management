# Product Management System
##  Features : 
**Add, Edit, Delete Products** 

###  Backend:
## technologies used :
1.Node.js,PostgreSQL
2.Express.js,Sequelize ORM,
### Frontend
## technologies used :
1.React.js ,Axios (for API requests)
2.Bootstrap 5
## how to run the system :
### Clone the repository
`git clone https://github.com/ibeshaho-amini/product-management.git
cd product-management`

### Setup Backend
`cd backend
npm instal`

### Run Sequelize migrations
`npx sequelize-cli db:migrate`
### Setup Frontend
`cd ../frontend
npm install
npm start`
